# 设计文档

这是一个用python做的大模型辅助工具。具体交互如下。

## UI/UX

### 截图输入

同时按下 Ctrl Shift 和左键进入此模式。

- 在屏幕显示一个细框矩形标记当前的框选位置
- 松开鼠标/松开快捷键确认截图

### 输出窗口

- 透明，只有文字悬浮（可以带shadow提升可读性）
- 不要有背景
- 不要有标题栏
- 不要在任务栏显示图标（静默的弹出）
- 按下 Ctrl Shift Z 切换显示/隐藏状态
- 聚焦时，按 Alt + 鼠标可以移动位置、调节大小
- 聚焦时，可直接框选内容

### 总体要求

- 使用 systray ，但是用户可配置是否显示 tray 图标

## 逻辑

1. 截图输入后，依据配置的模型判定是否需要使用OCR
2. 如果模型不带有视觉能力，调用OCR先识别文字
3. 得到文字/图片，填入提示词模板并发送
4. 流式接收模型答复，推到答案buffer
5. 输出窗口只负责把buffer显示出来，不主动清空buffer。不同回复之间使用横线隔开
6. 输出窗口不显示任何发送的提示词（只显示模型回复）

### 记忆管理

维护一个模型对话 session ，依据配置的 context 主动压缩。逻辑为

- 预留30%作为输出缓冲，即如果模型达到 70% 上下文则强制压缩

### 工具调用

#### 检索

在 `knowledge/` 目录下用户可存放纯文本文件。构建一个 function_call （如grep） 以支持模型调用工具来精确查询内容。
